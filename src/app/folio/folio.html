Some exercises completed for Computer Graphics 1 Spring of 2015

(behavior of these examples may vary from browser to browser)

View this repository on GitHub

  • Colors- uses HTML5 Canvas to approximate something similar to a simplified MS Paint
  • Prism- uses HTML5 SVG to build prisms basd on entered values
  • Polygon Circle Approximation- uses HTML5 Canvas to construct circle approximations from an arbitrary number of line segments
  • Midpoint Algorithm Shapes- Draws lines and shapes using an implementation of the Midpoint Algorithm. Some shape implementations work better than others.
  • SVG Basic Projections- Top, Front and Side views of a house with some customizable colors.
  • SVG More Projections- (from top left) Front, Side, 3-point Projection, 2-point Projection, 1-point Projection (from bottom left) Top, Isometric Projection, Trimetric Projection, Dimetric Projection, Oblique Projection
  • Trying out some 3D- An experiment with Three.js and materials. (from left) Reflective cube, die with texture and bump map, die with bump map only
